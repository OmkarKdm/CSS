CSS Solved a Big Problem

HTML was NEVER intended to contain tags for formatting a web page!

  • Mixing content and presentation in HTML made code maintainability difficult.
  • Ensuring consistency in design across a website was challenging.
  • Accessibility for screen readers and assistive technologies was hindered.
  • Loading times increased due to inline styles and repetitive tags.

CSS solved the separation of concerns, keeping content and presentation separate.

  • Consistent styles could be applied across multiple pages using external CSS files.
  • Accessibility improved by separating content from presentation.
  • Loading times reduced through cached external CSS files.
